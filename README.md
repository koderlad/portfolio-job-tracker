# Portfolio Project: Job Application Tracker

## Skills Acquired and Demonstrated
TypeScript, NextJS, Supabase, PostgreSQL, API Integration

## Problem
As an expat in Germany, one of the biggest challenge that I face to sustain in a completely new country and system is to find a suitable job. While the market is tough, unmanaged job application makes it even tougher. Applying to 100s of jobs and not having any of the data stored to find what roles am I getting calls for, what roles have I attended interview for and what companies have I been rejected before creates bigger problems and hinders my resume and approach improvements.

## Solution
Clarity gives direction, having a clear record and data to visualize the progress makes of job application makes watching progress incredibly effective and intuitive. With this goal in mind, I have pursued to build this project.

## Phases
### Phase 1 (Current)
Building a simple job application tracker (CRUD) using NextJS and Supabase.
